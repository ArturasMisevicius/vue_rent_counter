# Suggested commands
- Install PHP and JS dependencies: `composer install` and `npm install`
- One-shot setup: `composer run setup`
- Generate app key: `php artisan key:generate`
- Create SQLite database file if needed: `touch database/database.sqlite`
- Run migrations: `php artisan migrate`
- Start local app: `php artisan serve`
- Start frontend watcher: `npm run dev`
- Production frontend build: `npm run build`
- Run tests: `php artisan test --compact`
- Run a single test file: `php artisan test --compact tests/Feature/ExampleTest.php`
- Format PHP: `vendor/bin/pint --dirty`
- Refresh Composer autoload + package discovery when package state changes: `composer install` or `composer dump-autoload`
- Useful shell commands on Darwin: `git status`, `git diff`, `rg pattern path`, `find . -name`, `ls -la`, `cd <dir>`
