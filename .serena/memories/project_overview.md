Tenanto is a Filament-first Laravel multi-tenant utility billing and property management app. Roles/surfaces: SUPERADMIN control plane, ADMIN organization workspace, MANAGER restricted workspace, TENANT self-service portal. Stack verified in README and repo instructions: PHP ^8.3 (local 8.5.x), Laravel 13, Filament 5, Livewire 4, Tailwind 4, Vite 7, Pest 4, PHPUnit 12, SQLite default local DB. Important foundations: request validation in app/Http/Requests, mutations in app/Filament/Actions, shared support/read-model logic in app/Filament/Support, Eloquent only, Blade/Livewire frontend, no raw SQL outside strict model-internal allowances. Key modules include organizations, subscriptions, buildings, properties, tenants, meters, invoices, KYC, reports, translations, audit/security controls.
