# iOS Platform Guidance

## Defaults
- Primary font: SF Pro / SF Compact
- Min touch target: 44pt x 44pt
- Back navigation: edge swipe left
- Action sheet: bottom sheet
- Progress: spinner
- Pull to refresh: UIRefreshControl

## Guidance
- Follow Apple Human Interface Guidelines.
- Respect safe areas and large titles where appropriate.
- Use standard system controls unless there is a strong reason to diverge.
