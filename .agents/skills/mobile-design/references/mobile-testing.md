# Mobile Testing

## Testing pyramid
- Unit tests for business logic and utilities.
- Integration tests for data flows and offline sync.
- E2E tests for critical user journeys.

## Platform coverage
- Test on iOS and Android if cross-platform.
- Include low-end devices and older OS versions.
- Validate accessibility labels and touch target sizes.

## Scenarios to cover
- Offline start, reconnect, and conflict resolution.
- Slow network and high latency.
- Background / foreground transitions.
- Push notification deep linking.
