# Android Platform Guidance

## Defaults
- Primary font: Roboto
- Min touch target: 48dp x 48dp
- Back navigation: system back button/gesture
- Action sheet: bottom sheet or dialog
- Progress: linear progress or spinner
- Pull to refresh: SwipeRefreshLayout

## Guidance
- Follow Material Design 3.
- Respect system back behavior on every screen.
- Use standard system controls unless there is a strong reason to diverge.
